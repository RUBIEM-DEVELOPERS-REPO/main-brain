## Managing an Executive Calendar with Deep‑Work Blocks and Time‑zone Awareness  

| Step | Action | Rationale |
|------|--------|-----------|
| **1. Define deep‑work windows** | • Ask the executive which daily periods they want reserved for focused work (e.g., 9 am‑11 am, 2 pm‑4 pm). <br>• Create recurring events titled *Deep‑Work* (or similar) and colour‑code them. | Executives need uninterrupted time to think strategically; marking the slots as busy signals this need to colleagues [1]. |
| **2. Block first, schedule later** | • Set the deep‑work events to **busy** rather than “available”. <br>• Align the calendar’s *working‑hours* setting with those blocks so automated schedulers only propose times outside them. | When the calendar shows the period as busy, meeting‑request tools automatically avoid it, preserving focus. |
| **3. Use layered calendars** | • Keep a **primary calendar** for meetings and a **secondary “focus” calendar** for deep‑work. <br>• Show only the primary view for quick daily checks; reveal the secondary when planning longer‑term. | Separating calendars makes overload obvious while still keeping protected time visible. |
| **4. Add explicit time‑zone data** | • Set the correct time‑zone field for every event (city or UTC offset). <br>• For recurring meetings that involve participants in different zones, include the zone in the title (e.g., “Team Sync – EST”). | Explicit zones prevent mis‑alignment when attendees are distributed, ensuring the executive’s deep‑work blocks remain intact. |
| **5. Review and adjust weekly** | • At the start of each week, verify that no meetings have been inadvertently placed inside deep‑work slots. <br>• Adjust recurring events if the executive’s priorities shift. | Regular review catches scheduling drift and maintains the integrity of focus periods. |
