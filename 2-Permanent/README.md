# Permanent notes

The body of the brain. One atomic claim per note. Stable IDs. Dense linking.

## Filename
`YYYYMMDDHHmm-<slug>.md` — timestamp ID never changes once assigned.

## Frontmatter
```
---
id: 202604251430
title: <claim or insight, complete sentence>
tags: [<topic>, <topic>]
created: 2026-04-25
---
```

## Body shape
1. **First sentence** — the claim, standalone and intelligible.
2. **Body** — evidence, reasoning, examples; rephrased in your own words.
3. **`## Links`** — list of `[[id-slug]]` references with a one-line reason for each link. Every note links to at least one other; an orphan is a draft.

## Rules
- Atomicity: one idea per note. "And" in the title means split it.
- Don't paraphrase a source — paraphrase the *idea* and connect it.
- A note worth keeping is one you'd write the same way again from scratch.
