# Research: As a Executive Assistant, manage calendar with deep-work blocks and timezone awareness

**How an Executive Assistant can build a calendar that protects deep‑work time and stays timezone‑aware**

| Step | What to do | Why it matters (for an executive) |
|------|------------|-----------------------------------|
| **1. Clarify the executive’s “deep‑work” windows** | • Ask the executive for their preferred daily blocks (e.g., 9 am‑11 am, 2 pm‑4 pm). <br>• Record these as *recurring “focus” events* in the calendar with a distinct colour or label (e.g., **Deep‑Work**). | Executives are “people in a high position who make decisions and put them into action” — they need uninterrupted time to think strategically [1]. |
| **2. Block the time first, then schedule meetings around it** | • Create the deep‑work events as **“busy”** (not “available”). <br>• Turn on the calendar’s *“working hours”* setting to match those blocks, so meeting‑request tools only suggest slots outside them. | When the calendar shows the block as busy, colleagues’ scheduling tools will automatically avoid it, protecting the executive’s focus. |
| **3. Use multiple calendars or layers** | • Keep a **primary calendar** for meetings and a **secondary “focus” calendar** for deep‑work. <br>• Enable the “show only primary” view for quick daily scans, but keep the secondary visible when planning longer‑term. | Separating the two makes it easy to see at a glance whether a day is overloaded, while still preserving the visibility of protected time. |
| **4. Add timezone context to every event** | • When creating a meeting, always set the **timezone** field (most calendar apps let you pick a city or UTC offset). <br>• For recurring events, use the *“same time in each participant’s timezone”* option rather than a fixed UTC hour. | Executives often travel or work with global teams; explicit timezone data prevents accidental double‑booking across regions. |
| **5. Automate conversion & reminders** | • Enable **automatic timezone conversion** in the calendar settings so the event shows in the viewer’s local time. <br>• Add a **15‑minute “prep” reminder** that also displays the meeting’s timezone (e.g., “Prep for London call – 09:00 BST”). | A reminder that includes the timezone cue helps the executive and attendees double‑check the correct slot, especially when daylight‑saving changes occur. |
| **6. Share the deep‑work policy with the organization** | • Publish a short “calendar etiquette” note (e.g., “Please respect the blue‑shaded Deep‑Work blocks unless it’s an emergency”). <br>• Pin the note in the team’s shared drive or Slack channel. | Transparency reduces the number of last‑minute requests that try to intrude on focus time. |
| **7. Review & adjust weekly** | • At the end of each week, run a quick audit: <br>  – How many deep‑work blocks were kept? <br>  – Any conflicts caused by timezone errors? <br>  – Adjust the recurring pattern if the executive’s workload shifts. | Continuous refinement ensures the calendar stays aligned with the executive’s evolving priorities. |

### Quick‑reference template (Google Calendar / Outlook)

```
Title: Deep‑Work – Strategic Planning
When: Mon‑Fri 09:00‑11:00 (Executive’s local TZ)
Repeat: Weekly on weekdays
Visibility: Private
Show as: Busy
Color: #4A90E2 (blue)
```

- **Add a secondary “Time‑zone” field** (e.g., “TZ: America/New_York”) for clarity.
- **Enable “Show time‑zone”** in the calendar view so the executive can see at a glance which zone each block belongs to.

### Tools that help

| Tool | Feature that supports deep‑work & timezone |
|------|--------------------------------------------|
| **Google Calendar** | “World clock” sidebar, per‑event timezone, “working hours” setting. |
| **Microsoft Outlook** | “Time‑zone drop‑down” on event creation, “Focused Inbox” for meeting requests. |
| **Calendly / x.ai** | Can enforce “no‑booking during blocked times” and automatically convert invitee timezones. |

---

**Bottom line:** By defining recurring, colour‑coded deep‑work blocks, marking them as busy, and always attaching explicit timezone data to every meeting, an Executive Assistant creates a calendar that both shields the executive’s concentration and eliminates cross‑timezone mishaps. This systematic approach aligns with the executive’s need for decisive, uninterrupted work [1].
