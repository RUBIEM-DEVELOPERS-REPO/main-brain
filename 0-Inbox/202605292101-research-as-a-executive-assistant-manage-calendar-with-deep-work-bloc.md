# Research: As a Executive Assistant, manage calendar with deep-work blocks and timezone awareness.

Context: Role: Custom
Title: Manage calendar with deep-work blocks and timezone awareness
Description: Persona-derived starter task for "Evie"

**How an Executive Assistant (EA) can run a calendar that protects deep‑work time and stays timezone‑smart**

| Step | What to do | Why it matters (executive‑focused) |
|------|------------|-----------------------------------|
| **1. Capture the executive’s “focus rhythm”** | Ask the executive for their preferred deep‑work windows (e.g., 2 h × 2 days per week). Record them as *recurring “focus blocks”* in the calendar with a distinct colour (e.g., dark blue) and a label such as **🟦 Deep‑Work**. | Executives are “high‑position decision‑makers” who need uninterrupted time to think strategically [1]. Marking the blocks visually signals to everyone that the time is protected. |
| **2. Block out “buffer zones”** | Insert 10‑15 min buffers before and after each deep‑work block. Use the same colour but a lighter shade, and label them **Buffer**. | Buffers absorb meeting‑overrun spill‑over and give the executive a mental reset before diving back into focused work. |
| **3. Set “meeting‑type defaults”** | Create two meeting‑type templates: <br>• **Deep‑Work (Busy)** – auto‑decline invites that conflict. <br>• **Open (Free)** – for routine check‑ins that can be moved. | A clear default reduces the chance that ad‑hoc requests will intrude on focus time. |
| **4. Use timezone‑aware scheduling tools** | • When creating an event, always select **“Time zone”** and set it to the executive’s primary location. <br>• Enable the calendar’s *“Show time zones”* column so you can see the start/end in both the executive’s and participants’ zones. <br>• For cross‑regional meetings, propose 2–3 time‑slot options that fall within the executive’s “working hours” window (e.g., 9 am‑6 pm local). | Executives often work with global teams; explicit timezone handling prevents accidental off‑hours invites. |
| **5. Automate conflict resolution** | • Turn on *“Automatically decline conflicting events”* for the Deep‑Work calendar. <br>• Set a rule that any meeting request that lands inside a Deep‑Work block triggers an automatic *“Propose new time”* reply with the buffer‑adjusted alternatives. | Automation enforces the deep‑work policy without manual triage. |
| **6. Communicate the policy** | Draft a short “Calendar Etiquette” note (1‑2 sentences) and pin it to the executive’s calendar description: <br>> *“🟦 Deep‑Work blocks are protected for strategic thinking. Please use the “Propose new time” option for any conflict.”* | Transparency lets colleagues know the rationale and reduces friction. |
| **7. Review weekly** | Every Friday, scan the upcoming week: <br>• Confirm that deep‑work blocks remain untouched. <br>• Adjust any recurring meetings that repeatedly clash. | Regular review keeps the schedule aligned with shifting priorities. |

### Quick‑reference cheat sheet for the EA

| Action | Shortcut (Google Calendar) | Result |
|--------|----------------------------|--------|
| Create deep‑work block | `c` → *More options* → *Time zone* → set → *Busy* → colour **dark blue** | Protected focus time |
| Add buffer | Duplicate block, shorten by 10 min, colour **lighter blue**, label **Buffer** | Cushion for overruns |
| Propose new time (when conflict) | Open invite → *Suggest new time* → pick slot outside deep‑work | Keeps focus intact |
| Show multiple zones | Settings → *Show secondary time zone* | Instant visual of participant times |

**Bottom line:** By turning deep‑work periods into immutable, colour‑coded calendar entries, adding buffers, and leveraging timezone‑aware features, an EA can guarantee that the executive’s strategic thinking time stays sacrosanct while still accommodating global collaboration.
