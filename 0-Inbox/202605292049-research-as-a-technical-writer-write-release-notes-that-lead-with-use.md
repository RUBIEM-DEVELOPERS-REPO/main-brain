# Research: As a Technical Writer, write release notes that lead with user-facing benefit. Context: Role: Custom

**Release Notes – What’s in It for You?**  

- **Get work done faster** – The new batch‑processing engine automatically groups similar tasks, so you spend less time queuing work and more time seeing results.  
- **Never miss a deadline** – Real‑time notifications now surface in the system tray and via email, keeping you informed the moment a task completes or a dependency changes.  
- **Simpler onboarding** – A refreshed “Getting Started” wizard walks new users through the core concepts with plain‑language explanations, avoiding the dense, **technical** jargon that can slow learning (Cambridge Dictionary defines *technical* as “relating to the knowledge and methods of a particular subject or job” [1]).  
- **More control, less hassle** – Expanded permission settings let admins grant exactly the rights each team member needs, reducing accidental changes and improving security.  
- **Better visibility** – New dashboard widgets show live progress bars and performance metrics, so you can instantly gauge project health without digging through logs.  

**Detailed Changes**  

| Category | Change | Benefit |
|----------|--------|---------|
| **Batch Processing** | Auto‑grouping of similar tasks | Reduces manual setup time |
| **Notifications** | System‑tray & email alerts | Immediate awareness of status updates |
| **Onboarding** | Guided wizard with non‑technical language | Faster ramp‑up for new users |
| **Permissions** | Granular role‑based access control | Fewer accidental edits, tighter security |
| **Dashboard** | Live progress bars & metrics widgets | Quick health checks at a glance |

**Known Issues**  

- Some legacy integrations may require a one‑time re‑authentication after the permission overhaul.  
- Dashboard widgets may not display on screens with a resolution lower than 1024 × 768.

**Upgrade Instructions**  

1. Stop the NeuroWorks service.  
2. Run the installer and select “Upgrade”.  
3. Restart the service; the new dashboard appears automatically.  

For any questions, contact support at support@neuroworks.local.
