# Research: As a Job Summary, work with agents and brokers to gather additional information

**Job Summary – “Work with agents and brokers to gather additional information”**

| Aspect | Details (from the NeuroWorks vault) |
|--------|--------------------------------------|
| **Primary Goal** | Act as the liaison between internal teams and external agents/brokers, collecting any missing data required to move a transaction or project forward. |
| **Key Responsibilities** | 1. Identify gaps in the current information set (e.g., client details, policy terms, market data). <br>2. Reach out to agents and brokers via email, phone, or the NeuroWorks messaging hub to request the needed items. <br>3. Log each request and its status in the project’s knowledge vault. <br>4. Verify the accuracy and completeness of received information before passing it on to downstream workflows. |
| **Typical Tasks** | • Draft and send templated request messages. <br>• Track responses in the **_knowledge/neuroworks‑platform.md** task board. <br>• Update the “Research” notes with newly acquired data. <br>• Escalate overdue items to a supervisor or the “Escalations” team. |
| **Tools Used** | • **Clawbot** – the NeuroWorks multi‑persona assistant that can auto‑populate request emails and set reminders. <br>• **Vault notes** – store all correspondence and evidence of received information. <br>• **Parallel team dispatch** – assign follow‑up subtasks to other agents if the broker’s response requires specialist input. |
| **Success Metrics** | • % of information gaps closed within the SLA (usually 48 h). <br>• Reduction in back‑and‑forth clarification cycles. <br>• Accuracy rating of the gathered data (validated against source documents). |
| **Dependencies** | • Access to the broker/agent contact directory stored in the NeuroWorks vault. <br>• Clear definition of what constitutes “additional information” for each project (often outlined in the initial brief). |
| **Common Challenges** | • Delayed responses from external parties. <br>• Inconsistent data formats (e.g., PDFs vs. spreadsheets). <br>• Need to reconcile conflicting information from multiple brokers. |
| **Best Practices** | 1. Use the **standard request template** saved in the vault to ensure consistency. <br>2. Log every outreach attempt (date, channel, outcome). <br>3. Follow up politely but promptly (24 h after the first request). <br>4. When possible, attach a short “what we need” checklist to make the request clear. |

**How to Execute the Summary in NeuroWorks**

1. **Open the job ticket** in the NeuroWorks dashboard (`_neuroworks/jobs/2026-05-29‑work‑with‑agents‑and‑brokers‑to‑gather‑additional‑information…md`).
