# Research: what is firecrawl

**Firecrawl** is a developer‑focused web‑data platform that lets AI applications **search, scrape, and interact with live web pages at scale**.  

- It provides an API (and a CLI / Python/Node.js SDK) that can:
  * **Search** the web and return full‑content results.  
  * **Scrape** any URL and deliver the data in LLM‑ready formats such as Markdown, JSON, screenshots, etc.  
  * **Interact** with pages (click, navigate, run code) via AI prompts.  
- The service is marketed as the “infrastructure layer” for AI agents, enabling them to fetch up‑to‑date web information without building their own crawling logic.  
- Firecrawl is **open‑source** (GitHub repo: github.com/firecrawl/firecrawl) and is already used by dozens of companies (Shopify, Canva, Zapier, Replit, etc.) to power chatbots, agents, and other AI products.  

In short, Firecrawl is a cloud‑based, open‑source API that abstracts web crawling, cleaning, and formatting so AI systems can easily obtain reliable, real‑time web data.
