# Research: In one sentence, what is a knowledge pack

A Knowledge Pack is a structured, business‑specific collection of reference documents, files, or web content that can be attached to an agent’s inbound or outbound configuration so the AI can retrieve accurate, context‑specific information during interactions, but it should not contain behavioral instructions or scripts [1].
