# Research: As a Product Manager, write release notes that name what changed for the customer, not what was shipped

**Release Notes – What Changes For You**

| What the customer experiences | Why it matters |
|------------------------------|----------------|
| **Faster task‑dispatch** – commands you send to the AI‑workforce are now routed through an optimized queue, cutting average response time by ~30 %. | You get results sooner, so multi‑step workflows finish quicker and you can keep projects moving without waiting on the system. |
| **Unified “Team” view** – a new sidebar shows all active personas (e.g., Researcher, Scheduler, Email‑Bridge) in one place, with status indicators and a single click to switch. | No more hunting through separate windows; you can see who is working on what and re‑assign tasks on the fly, improving oversight and collaboration. |
| **Scheduled‑task preview** – before a task is auto‑run, the UI displays a short preview of the generated prompt and expected output. You can approve, edit, or cancel. | Gives you confidence that automated runs will do exactly what you need, reducing the risk of unwanted actions. |
| **Email‑bridge enhancements** – inbound emails are now auto‑tagged with the relevant persona and attached to the correct project thread. | Your inbox stays organized; you no longer need to manually forward or rename messages to keep the AI‑workforce in sync. |
| **Improved error handling** – when a command fails, the system now returns a clear, actionable message (e.g., “Missing parameter X”) instead of a generic “error”. | Saves you time troubleshooting and lets you correct issues instantly, keeping work flowing. |
| **Local‑only mode toggle** – you can now lock the platform to 127.0.0.1 only, preventing any accidental external connections. | Enhances privacy and security for sensitive projects, giving you full control over where data travels. |

### How to see the changes
1. Open the **NeuroWorks Dashboard** – the new “Team” pane appears on the left.  
2. Run any command; you’ll notice the quicker response and the preview screen for scheduled tasks.  
3. Check your email client – incoming messages now carry the AI‑persona tag automatically.  

These updates are all about **what you, the customer, will notice**: speed, clarity, control, and security. No technical jargon about the underlying code or libraries is required – just the benefits you’ll feel in your daily workflow.
