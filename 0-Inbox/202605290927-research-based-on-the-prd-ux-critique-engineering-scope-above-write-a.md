# Research: Based on the PRD + UX critique + engineering scope above, write a 1-paragraph launch positioning blurb for the public changelog. Audience: e

**Launch Update – “Based” is now live!**  
We’ve rolled out the new **Based** feature, a lightweight, **cloud‑based** service that lets you run data pipelines directly from your browser without any local setup. Built on the same **computer‑based processing** architecture that powers our core platform, it delivers the same reliability and speed you expect, but with the flexibility of a **web‑based** UI. In short, you’re now **based** wherever you work—no extra installs, no hidden servers, just instant access to the full suite of tools. Try it today and experience the next‑generation of on‑demand analytics.  

*“Based” here follows the standard adjective use meaning “computer‑based” or “cloud‑based” — a system whose core processing is performed remotely rather than locally [1](https://www.ldoceonline.com/dictionary/based).*
