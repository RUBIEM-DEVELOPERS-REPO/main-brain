# Literature notes

One file per source (book, paper, talk, podcast, article).

## Filename
`<author-lastname>-<short-title>.md` — e.g. `ahrens-how-to-take-smart-notes.md`.

## Frontmatter
```
---
title: <full title>
author: <name>
year: <YYYY>
source: <url or citation>
status: reading | done
created: 2026-04-25
---
```

## Body
- Quote sparingly. Always rephrase in your own words — that's where understanding happens.
- For each idea worth keeping, write a permanent note in `../2-Permanent/` and link back here.
